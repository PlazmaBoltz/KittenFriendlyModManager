KittenFriendlyModManager (KFMM) is a mod management tool based in python that can install and enable mods for you!

-- Current Features --
- Downloading Mods from Web
- Installing Mods to KSA
- Enabling Mods to KSA

-- Planned Features --
- Removing and Disabling Mods
- Launching Options
- UI

-- How To Use --
1. Unzip installed file
2a. Open Unzipped Folder and Launch main.exe
2b. Alternatively you can run main.py if you have python installed
3. Have fun!

Let me know if you have any issues 😁
